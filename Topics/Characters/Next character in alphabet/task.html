Next character in alphabet

Hey there! This problem might be a bit unpredictable, but give it a go and let us know how you do!
Given a lowercase character, write a program that prints the next character in the alphabet. If the input character is 'z', print 'a'. Input: A single lowercase character. Output: The next character in the alphabet after the input character.
Sample Input:
a

Sample Output:
b


Sample Input:
m

Sample Output:
n



Memory limit: 256 MB
Time limit: 5 seconds

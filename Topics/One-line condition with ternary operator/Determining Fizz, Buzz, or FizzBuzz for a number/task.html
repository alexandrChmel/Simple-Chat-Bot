Determining Fizz, Buzz, or FizzBuzz for a number

Given a number as input, if the number is divisible by 5 print 'Fizz', if it is divisible by 3 print 'Buzz', and if it is divisible by both 3 and 5 print 'FizzBuzz'. In case none of these conditions are met print the input number. Write your code in one line using the ternary operator.


Sample Input:
10

Sample Output:
Fizz


Sample Input:
12

Sample Output:
Buzz



Memory limit: 256 MB
Time limit: 5 seconds

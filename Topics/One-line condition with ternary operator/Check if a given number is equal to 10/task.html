Check if a given number is equal to 10

Given a number, write a program in Java to determine whether this number is equal to 10 or not. Your program should take a single integer as input and print "Equal" if the number is 10, or "Not Equal" otherwise.


Sample Input:
10

Sample Output:
Equal


Sample Input:
20

Sample Output:
Not Equal



Memory limit: 256 MB
Time limit: 5 seconds
